# School Choice KZ

Современный русскоязычный сайт для родителей, выбирающих школу для ребенка.

## Стек

- Next.js 15
- TypeScript
- Tailwind CSS

## Страницы

- `/` — главная страница с hero, преимуществами, статистикой и футером
- `/catalog` — каталог школ с поиском, фильтрами и карточками
- `/schools/[id]` — детальная страница школы
- `/about` — страница о проекте

## Запуск

```bash
npm install
npm run dev
```
